⚡ JavaScript - Linguagem de Programação

document.getElementById()
Seleciona elemento pelo ID
const elemento = document.getElementById('meuId'); elemento.textContent = 'Novo texto';
document.querySelector()
Seleciona primeiro elemento que corresponde ao seletor
const elemento = document.querySelector('.classe'); const botao = document.querySelector('#botao');
addEventListener()
Adiciona ouvinte de evento
elemento.addEventListener('click', function() { alert('Clicado!'); });
Array.map()
Cria novo array transformando cada elemento
const numeros = [1, 2, 3, 4]; const dobrados = numeros.map(n => n * 2); // [2, 4, 6, 8]
Array.filter()
Filtra elementos baseado em condição
const numeros = [1, 2, 3, 4, 5]; const pares = numeros.filter(n => n % 2 === 0); // [2, 4]
Array.forEach()
Executa função para cada elemento
const frutas = ['maçã', 'banana', 'laranja']; frutas.forEach(fruta => { console.log(fruta); });
String.split()
Divide string em array
const frase = "olá mundo"; const palavras = frase.split(" "); // ["olá", "mundo"]
String.includes()
Verifica se string contém substring
const texto = "JavaScript é incrível"; const contem = texto.includes("Script"); // true
preventDefault()
Previne comportamento padrão do evento
form.addEventListener('submit', function(e) { e.preventDefault(); // Processa form sem recarregar página });
createElement()
Cria novo elemento HTML
const div = document.createElement('div'); div.textContent = 'Novo elemento'; document.body.appendChild(div);
classList
Manipula classes CSS do elemento
elemento.classList.add('ativa'); elemento.classList.remove('inativa'); elemento.classList.toggle('visivel');
Array.reduce()
Reduz array a um único valor
const numeros = [1, 2, 3, 4]; const soma = numeros.reduce((acc, n) => acc + n, 0); // 10

📚 Glossário - Termos Essenciais

Entenda os conceitos fundamentais do desenvolvimento front-end

Array
Uma estrutura de dados que armazena múltiplos valores em uma única variável, organizados por índices (posições). É como uma lista ordenada de elementos.
const frutas = ['maçã', 'banana', 'laranja']; console.log(frutas[0]); // 'maçã' console.log(frutas.length); // 3
Exemplo prático: Uma lista de tarefas, galeria de fotos, carrinho de compras
String
Tipo de dado que representa texto. É uma sequência de caracteres entre aspas simples (''), duplas ("") ou crases (``).
const nome = "João"; const sobrenome = 'Silva'; const nomeCompleto = `${nome} ${sobrenome}`; console.log(nomeCompleto); // "João Silva"
Exemplo prático: Nomes, endereços, mensagens, títulos de páginas
DOM
Document Object Model - Representação estruturada do HTML que permite ao JavaScript acessar e modificar elementos da página em tempo real.
// Acessar elemento const titulo = document.getElementById('titulo'); // Modificar conteúdo titulo.textContent = 'Novo título'; // Adicionar evento titulo.addEventListener('click', função);
Exemplo prático: Mudar texto ao clicar, validar formulários, criar animações
Função
Bloco de código reutilizável que executa uma tarefa específica. Pode receber parâmetros e retornar valores.
// Declaração de função function somar(a, b) { return a + b; } // Chamada da função const resultado = somar(5, 3); // 8
Exemplo prático: Calcular preços, validar dados, enviar formulários
Variável
Container que armazena dados/valores que podem ser utilizados e modificados durante a execução do programa.
let idade = 25; // Pode ser alterada const nome = "Ana"; // Constante (não muda) var cidade = "SP"; // Forma antiga (evitar) idade = 26; // OK // nome = "João"; // ERRO!
let: valores que mudam | const: valores fixos
Evento
Ação que acontece na página (clique, digitação, movimento do mouse) que pode disparar código JavaScript.
// Evento de clique botao.addEventListener('click', function() { alert('Botão clicado!'); }); // Evento de digitação input.addEventListener('input', verificarTexto);
Tipos: click, submit, keypress, mouseover, load, scroll
Seletor
Padrão usado para selecionar elementos HTML que receberão estilos CSS. Define "onde" aplicar as regras de estilo.
/* Seletor de tag */ h1 { color: blue; } /* Seletor de classe */ .destaque { background: yellow; } /* Seletor de ID */ #cabecalho { font-size: 24px; }
Tipos: tag, .classe, #id, [atributo], :hover
Propriedade
Característica de estilo que você quer modificar em um elemento (cor, tamanho, posição, etc.). Sempre seguida de dois pontos (:) e um valor.
p { color: red; /* propriedade: valor */ font-size: 16px; /* propriedade: valor */ margin: 10px; /* propriedade: valor */ }
Exemplos: color, font-size, background, margin, padding, border
Box Model
Conceito que define como os elementos HTML ocupam espaço na página. Cada elemento é uma "caixa" com conteúdo, padding, border e margin.
div { width: 200px; /* largura do conteúdo */ padding: 20px; /* espaço interno */ border: 2px solid; /* borda */ margin: 10px; /* espaço externo */ }
Ordem: Conteúdo → Padding → Border → Margin
Flexbox
Sistema de layout CSS que facilita o alinhamento e distribuição de elementos em uma linha ou coluna de forma flexível.
.container { display: flex; justify-content: center; /* alinha horizontal */ align-items: center; /* alinha vertical */ flex-direction: row; /* direção dos itens */ }
Uso: Centralizar conteúdo, criar layouts responsivos, organizar elementos
Tag
Elemento HTML delimitado por colchetes angulares (< >) que define a estrutura e o significado do conteúdo na página.
<h1>Título Principal</h1> <p>Parágrafo de texto</p> <img src="foto.jpg" alt="Descrição"> <a href="link.html">Link</a>
Tipos: Tags de abertura/fechamento, tags auto-fechantes
Atributo
Informação adicional fornecida dentro da tag de abertura que modifica o comportamento ou aparência do elemento.
<img src="imagem.jpg" alt="Descrição" width="300"> <a href="https://google.com" target="_blank"> <div class="container" id="principal">
Comuns: src, href, class, id, alt, width, height
Elemento
Componente completo formado por tag de abertura, conteúdo e tag de fechamento (quando aplicável). É a unidade básica do HTML.
<!-- Elemento completo --> <h1 class="titulo">Meu Título</h1> <!-- Elemento auto-fechante --> <br> <hr>
Partes: Tag abertura + Conteúdo + Tag fechamento
Front-end
Parte de uma aplicação web que o usuário vê e interage diretamente. Inclui design, interface, experiência do usuário e interações.
Tecnologias principais: - HTML (estrutura) - CSS (aparência) - JavaScript (interatividade) - Frameworks: React, Vue, Angular
Oposto: Back-end (servidor, banco de dados, lógica interna)
API
Application Programming Interface - Conjunto de regras e protocolos que permite diferentes aplicações se comunicarem entre si.
// Exemplo de uso de API fetch('https://api.exemplo.com/dados') .then(response => response.json()) .then(dados => console.log(dados));
Exemplo: Buscar dados do clima, integrar com redes sociais, pagamentos
Framework
Conjunto de ferramentas, bibliotecas e convenções que facilita o desenvolvimento, fornecendo uma base estruturada para construir aplicações.
Exemplos populares: - React (JavaScript) - Bootstrap (CSS) - jQuery (JavaScript - biblioteca) - Tailwind CSS (CSS)
Vantagem: Acelera desenvolvimento, código mais organizado
Responsivo
Design que se adapta automaticamente a diferentes tamanhos de tela (desktop, tablet, celular), proporcionando boa experiência em qualquer dispositivo.
/* Media Queries para responsividade */ @media (max-width: 768px) { .container { width: 100%; padding: 10px; } }
Técnicas: Media queries, Flexbox, Grid, unidades relativas (%, em, rem)
Debugging
Processo de encontrar e corrigir erros (bugs) no código. Envolve testar, analisar e identificar problemas na aplicação.
// Ferramentas de debug console.log(variavel); // Mostrar valores console.error(erro); // Mostrar erros debugger; // Pausar execução // F12 no navegador = DevTools
Ferramentas: Console do navegador, DevTools, breakpoints
